ЗАДАЧА1

Написать программу Калькулятор. Ввод данных и вывод результатов осуществить в одном и том же текстовом поле. Калькулятор должен содержать кнопки для ввода чисел, арифметических операций, кнопку для удаления последнего введеного символа и др.







ЗАДАЧА2

Написать программу Матричный калькулятор, который выполняет две операции - сложение и умножение двух матриц. Программа должна запросить у пользователя число n - размерность матрицы (квадратной). И по заданному пользователем n вывести на страницу две формы для ввода чисел в матрицы размерностью n*n и форму для вывода результата- тоже матрицу размерностью n*n.

ЗАДАЧА3

Написать программу, проверяющую корректность введенных пользователем данных в регистрационную форму. Должны быть рассмотрены следующие варианты: все поля заполнены, имя состоит только из допустимых символов, поле e-mail содержит @ и точку, поле пароль состоит только из латинских букв и цифр, и обязательно содержит хотя бы одну цифру, длина пароля не менее 7 символов, совпадение пароля и повторного пароля.

ЗАДАЧА9

Определить вибрирующее число по введенной дате рождения. Вибрирующее число в нумерологии – это сумма цифр, расположенных в дате рождения. Если в результате будет двузначное число, то для него снова вычисляется сумма цифр и так до тех. пор, пока не получим однозначное число. (Например, 11.07.1999. 1+1+0+7+1+9+9+9=37 3+7=10 1+0=1 – вибрирующее число).

ЗАДАЧА10

Реализовать игру в числа. Компьютер “загадывает” четырехзначное число с неповторяющимися цифрами и не начинающееся с нуля. Пользователь должен за десять ходов угадать это число. После каждого хода компьютер анализирует число введенное пользователем и выдает результат в виде: 3:2 - это означает, что 3 цифры угадано, но только две из них стоят на своих местах. Если пользователь не угадал число полностью за десять ходов – то он проиграл. Данные и результаты всех ходов текущей игры отображать на странице до.
